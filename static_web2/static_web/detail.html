文化进万家 踏歌起舞迎新年

发布时间：2015-09-09   作者：Shane   浏览量：989

桃芳杏菲催人进，踏歌起舞迎新年。昨日上午，市文化馆广场上曲乐悠扬，掌声连连，2016“我们的中国梦”——文化进万家专场演出正隆重举行。为深入开展“深入生活扎根人民”主题实践活动，进一步丰富群众精神文化生活，实现文化惠民、为民、乐民，推进文化强市建设，市文化委联合市文化馆、市民间文艺家协会及市群众文化学会举办了此项活动。文化进万家首场演出从昨日起至3日将连续举行六场，演出内容丰富、形式多样，受到了在场观众的热烈欢迎。“好”、“再来一个”……女声小合唱《祖国万岁》拉开了演出的序幕，精彩的表演让观众不断叫好。伴随着舞蹈者翩然的身姿，由汀棠办事处艺术团选送的京剧选段《梨花颂》铿锵有力、意蕴悠长，把现场气氛推向了高潮。随后，器乐演奏、时装秀、男声小合唱也依次上演。经常参加文化惠民演出的孙桂梅表示很高兴参加此次活动：“能和老百姓分享快乐，把文化的乐趣传递给大家，我觉得很有意义。”整场节目精彩纷呈，赢得了观众的阵阵掌声。“在新年的第一天看到这么多艺术家的表演，心里很温暖。”60岁的王大爷说。据介绍，在后面的几场文艺演出中，全市广大文艺工作者还将继续为市民带来舞蹈、声乐、
